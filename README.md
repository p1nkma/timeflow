# TimeFlow

Веб-приложение для тайм-менеджмента с Telegram Mini App как killer-фичей.
Цель — сократить временные потери пользователя через умное планирование, энергозоны и AI-советы.

---

## Структура монорепо

```
timeflow/
├── frontend/   React 19 + Vite + TypeScript
├── backend/    FastAPI + SQLAlchemy + PostgreSQL
└── tma/        Telegram Mini App (после полного бэкенда)
```

- [frontend/README.md](frontend/README.md) — фронт, как запускать
- [backend/README.md](backend/README.md) — бэк, как запускать
- CLAUDE.md — заметки по проекту для Claude Code (в gitignore)
